# AWS Interview questions

1. Your application latency suddenly increased in Production. How do you troubleshoot across ALB → EC2 → RDS → VPC flow?
2. You need zero-downtime deployment for an API running on EC2. Which strategy would you choose — Blue/Green or Rolling — and why?
3. How do you secure cross-account access for CI/CD pipelines using IAM roles?
4. Your Lambda function is hitting cold starts during spikes. How do you fix it?
5. How do you design a highly available VPC across 3 AZs with private subnets, NAT gateways, and on-prem connectivity?
